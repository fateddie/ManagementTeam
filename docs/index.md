## üîç Search Tips

This documentation portal supports **two types of search**:

1. **Keyword Search** - Traditional full-text search (top search bar)
   - Searches all documentation files
   - Highlights matching terms
   - Fast and familiar

2. **Semantic/Vector Search** - AI-powered semantic search (via MCP plugin)
   - Understands meaning, not just keywords
   - Example: Search "how to reduce costs" finds Cost Optimization docs
   - Powered by `all-MiniLM-L6-v2` embedding model

**Pro Tip:** Use dark mode toggle (top right) for comfortable nighttime reading! üåô
